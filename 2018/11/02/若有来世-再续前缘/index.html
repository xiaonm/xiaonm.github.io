若有来世,再续前缘

发表于 2018-11-02 | 分类于 Essay | | 浏览 次 | 字数： 601
阅读时长 ≈ 2

昨睡的早，半夜醒来，再难入眠。窗外无星无月，浓如墨。寒风像吹哨子似得凛冽着大地。寒意不言而喻，夜无边的黑使孤独渗入惆怅，思绪在夜空里随风肆意飘荡。想你了，没有理由没有准备如潮汐涌动，打湿了心田。一切的一切突如其来。
翻身查看手机，已经是凌晨3点，屋子里飘荡着舍友熟睡的呼吸声，那么的均匀，那么的安详，此刻一定魂牵梦绕，幽会漂亮的姑娘。怕惊扰邻里的好梦，轻轻的起身下床，蹑手蹑脚的来到桌旁倒一杯水，竟连清泉入杯也带着淡淡的忧伤。想你的美，你的笑，你的可爱，想当初的我们，就像甜甜的刚入唇的水，初恋般的感觉，甜甜的，美美的让人回味。看现在的你，现在的我们，如咽下去的水感觉全无，顷刻之间心凉飕飕的，令人绝望。早已预言有如今这样的结局，但看到你当时天真烂漫的笑容，温柔的话语，我是爱的那么纯粹，爱的那么彻底，毫无保留，纵使伤了，痛了，也不回头。明知仅仅是华丽的过程，悲催的结局，也愿意使自己沦陷。
窗外稍亮，穿衣轻声出门，晨跑在大学的操场上，一圈一圈却那么的漫无目的，此刻早已身心分离，此情此景正是我多年来恋爱的场景再现，一次次的义无反顾的投入，最终都无情的回到了原点，身心却变得憔悴。仰望天空，已是破晓时分，如水晶般剔透，又如秋水分明般媛儿眼眶里晶莹的泪水，让人心碎。忍不住想问：“你现在还好吗？“分手那时，当你走出房门轻轻带上那刻起，也将永远关闭了我们的宿命之门。也曾经告诉自己，既然离开就要干脆，从此不会再有任何联系，可现在，多么想知道你的消息，想知道你的一举一动，一悲一喜。。。。。
媛本缘，无缘与你相恋。爱就爱了，纵使葬送情海也不后悔。今生无缘相恋，来生还追你。。。那时你会嫁给我吗？？！！！
