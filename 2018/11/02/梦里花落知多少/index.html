梦里花落知多少

发表于 2018-11-02 | 分类于 Essay | | 浏览 次 | 字数： 804
阅读时长 ≈ 2

人就是这样，当有一场美丽雪景摆在你面前让你去描写时，你不懂得珍惜，等它被时间融化且只留一丝痕迹的时候才追悔莫及。记得那天早晨我在隔壁女孩的开门声中苏醒（注：是开她家的门，不要误会哦）便听到她对舍友说：“下雪了，好大好美哦。”‘啪’关上房门没了音讯，我突然感觉屋子里有一丝凉意。不禁哆嗦了一下。穿上衣服打开房门严寒便袭击而来，走出去一看，房顶白了，地也白了，灰蒙蒙的天空上鹅毛般的大雪纷纷降下，紧密却很均匀。仿佛一朵朵洁白的梨花花瓣从天而降，不知梨花被谁摘。一首诗便涌入心头“忽如一夜春风来千树万树梨花开”我就觉得我应该写点东西了。狠自己这些年没有能多喝几瓶墨水，在心里几次回旋，最终像宋丹丹写《月子》那样半天憋不出一个字来，所以告一段落。

从出租房到书店大概有半个小时的路程，每天我总是步行而至。一来就当锻炼身体；二来可以沿途观光（主要是美眉哦，呵呵）。今天也不例外，和往常一样洗簌完，屋子简单的收拾了一下就匆匆启程了，耳朵里还是插着耳机，还是听着郑源的悲伤情歌，这是我的必须，路上听他的歌已经有两年的历史，两年来他的歌声一直伴之我左右，分享着我的苦闷生活，与往常不一样的是今天特别的冷，寒风转肌入肤骨，出了巷口路上的行人已经很多了，却走的匆忙。雪花毫不留情地打到行人的头上、脸上、身上仿佛一个个活生生的雪人在死亡线上挣扎。路上很滑，每一步我都迈的认真，手里举着一把半旧的雨伞，面对强烈的冰花却无能为力，也难怪它是雨伞而不叫雪伞。

拐进迎泽大街映入眼帘的是一个琼楼玉宇的世界，很美，很新。时不时的看见行人举起手机记录着这一时刻，我无暇欣赏，却想：“有病，这么冷的天要是不上班还不如在家搂着老婆睡，站那拍照是为了显示自己的与众不同还是展示自己的高雅姿态？”想总归想，我还是管不着，著名诗人但丁的一句名言：“管他呢，走自己的路让别人去拍吧！”呵呵 。

好景不常在，第二天下午，云 像是青花瓷器上的裂痕一样支离破碎，蓝天隐隐可见，慢慢太阳露出了笑脸，全世界的雪花开始化为流水四散逃逸，门口不知谁家的雪人也热的满身大汗低头哈腰，终于天晴了，云散净了，整个城市的人们都投身于清理积雪的革命中来，从此梦里花落知多少？
