我是唐僧

发表于 2018-11-02 | 分类于 Essay | | 浏览 次 | 字数： 4k
阅读时长 ≈ 13

个人简历 ­

姓名：亮仔

字： 公貔貅

职业：和尚

法号：玄奘

成长经历： ­

公元后87年3月生于山西汉族自治区，3岁时拜妈妈为师学习汉族文化，由于天资聪慧，7岁时已经才高八斗，熟读绕口令及各类兵法经典，唐诗宋词倒背如流；让过目不忘的黄蓉姐姐也对我刮目相看时不时的逼我和她私定终生，导致郭 靖郭大侠悬梁自尽，含笑九泉，才使元兵趁机拿下了镶阳城。8岁时拜爸爸为师，学习摸爬滚打擒拿手，由于勤学苦练，15 ­岁时已经熟练各类武学经典，加上著名少林方丈禅师周杰龙 点化，在武林大会上技压群雄，尤其《降龙十八掌》《大力金刚腿》让各武林同道甘败下风，再加上有《九阳真经》神功护体所谓人者无敌， 所向披靡，领魔教教主张无忌和恒山令狐掌门丢尽脸面，无地自容，被迫双双自刎。16岁时拜姐姐为师学习琴棋书画，由于天赋盖身，17岁时创作出举世闻名的《蒙娜：丽莎》令才女秋香姑娘对我情意绵绵，寝食难安，伯虎兄接受不了服毒自尽吐血含冤而死。 ­

18岁时妈妈和我说：­

“孩子你长大了，该娶媳妇了。”­

“妈妈我还小，我不嫁人。”我撅着小嘴嚷道，一边按着电视遥控器。­

“妈妈我喜洋洋和灰太狼都没有看完呢！我不嫁人。”­

“傻孩子。”妈妈抚摸着我的头发笑着说。­

后天是妈妈的生日，她是想在生日那天让我把黄蓉和秋香一并娶进门，来个双喜临门。可我觉得我还小，尚未发育成熟，所以一再推辞，可母命难为，加上二位美女再三催命与我也早有婚约，我也就不在做声。­

结婚那天，红红火火，全家上下齐欢腾，花前月下，洞房花烛。­

酒席罢！二位夫人早已守候在床头，却因我年少无知，妈妈也没能提前给我买《新婚必读》系列丛书。让我面对二位娇妻却无从下口。­

黄蓉聪明伶俐，看我这般可爱，不禁笑出声来。我万分惊恐，落荒而逃，逃跑的身影歪歪扭扭，一溜烟的朝大门外跑出，且一夜未归，­

第二天一大早，我被街上喧哗的嘈杂声惊醒，看见人们成群结队朝天安门方向走去。心里疑惑，想凑个热闹便加入队伍中来，一路上横冲直闯，左厮右打，终于挤到队伍前面，这下让我傻了眼，带头大哥竟然是俺的两个老婆，由于对俺不满，便拉帮结派到天安门广场示威游行，表示强烈抗议并加以谴责，那架势似乎不扭掉俺的脑袋也要废了俺两天腿，­

这事非同小可，闹得京城上下沸沸扬扬，都想看俺的笑话。­

国务院总理兼军机大臣 和珅 早已对我恨之入骨，恨不得将我五马分尸；闻讯觉得机会来了，是铲除我的最佳时机；便深夜入宫，拜见太宗李世民：­

“陛下，臣和珅有事启奏。”­

“和爱卿不必多礼，有何等要事，劳深夜入宫”­

“陛下，亮仔不忠不孝、不仁不义；臣恳请圣上治其死罪。”­

“哦，有这等事？”皇上用手摸索着自己的胡须。“何以见得。”­

“皇上有所不知，今日亮仔之妻只因为家庭琐事，竟劳师动众到宫外天安门楼下，摇旗呐喊，全然不把皇上放在眼里，为不忠；父母希望其早日成家立业，生儿育女，有朝一日报效国家，可亮仔整日花天酒地，青楼妓院，最可恨的事天天看动画片，让父母大失所望，为不孝；新婚之夜不伴爱妻，竟离家出走一夜不归，令新娘独守空房，为不仁；气死郭靖夺其所爱，逼死恒山掌门令狐冲与魔教教主张无忌，为不义；一个不忠不孝不仁不义之人该杀，死不足惜。”­

和珅稍作停留见太宗没有言语，便追加道：­

“皇上，亮仔不忠，目无圣上，仅此一条就该株连九族啊 皇上，他武功盖世，但不见为国家杀一个倭寇，整日争强斗狠，使我大唐丧失了诸多英雄豪杰，国力大衰，周边诸侯才如此嚣张，几次犯我边疆，皇上此人罪恶滔天不可饶恕。”­

太宗觉得和爱卿言之有理，正在犹豫之时，只见大殿之上蓬荜生辉，佛光四射，来自一个天籁之音道：­

“和珅你休得胡言乱语，据本座所知你所言不实。今日之事是其妻所为，并非亮仔本意。陛下，亮仔是百年来难得的奇才，现有真经与西天天竺国大雷音寺内，取得真经方可修成正果，此人武功盖世，意志坚强，是拜佛求经的最佳人选，何不派往，将功补过呢？”­

话音刚落，佛光便渐渐暗淡消失在大殿之中。太宗知道方才那声音是观音菩萨圣临，便连忙跪别，得到菩萨点化便连夜宣召亮仔进宫面圣，和珅不敢做声，心想;”这样也好，从大唐道西天遥遥万里，一路上，妖魔鬼怪作祟，量他也难以保命。”­

魏公公前来宣旨，我听后起身就走，宛如慷慨赴义。­

杨贵妃早就听说我侠肝义胆，帅气潇洒，气死郭靖迷倒黄蓉，武林大会逼死令狐冲和张无忌。更是对我倾慕有加，朝思暮想，加上久居深宫，异性莫过于皇上太监，对我的爱自不待言，听说我连夜进宫面圣，便早已说服皇上，垂帘听政，早早就绪于帷幄之中。承诺：只远观，不近看；只听说，不言语；­

“草民亮仔叩见皇上，吾皇万岁万岁万万岁。”­

“亮仔朕听说你功高盖世，文韬武略，现在经书于天竺，你可愿意为朕前往求取真经？”­

“草民愿意前往”想都没有想便答道。­

“果然爽快，朕与你从此兄弟相称，尽享荣华富贵，朕还要把朕的爱妃玉环赠于你，你意下如何？”­

我一听杨玉环就热血澎湃，四大美女耶！！！！！！！我早就！！！！！呵呵。­

杨贵妃一听皇上把自己赠与眼前这位自己暗恋了许久的男人激动的热泪盈眶，眼睛瞪着圆溜，把心提到了嗓子眼等着我的回应！­

“皇上，草民乃一介武夫，躬耕于南阳，苟全性命于乱世，不求闻达于诸侯，陛下不以草民卑鄙，猥自枉屈，受命于草民，谘以当世之事，由是感激。只不过一介草民难以配的上千金之躯，，，，，。”­

“我愿意！！！！！！”没等我说完贵妃娘娘便迫不及待的从帷帐里走了出来，道：“如君不嫌弃，妾愿意与君长相思守，决不后悔！”说着走向前来将我扶起相拥而立。那触电般的感觉你们知道吗？？？？？

“哈哈哈！！朕早就看出来玉环她真正爱的人是你，朕老了，她还那么年轻，朕看你们情投意合，把她留在宫中委屈一生，倒不如成全一桩美事。哈哈哈”­

我此时还能说点什么呢？？？？？？

就这样俺们师徒四人。。。。不好意思，是妻妾四人，次日踏上了去往西天的路上，临行前太宗赐予我一匹白龙马和通关文叠。­

黄蓉自小就熟读武学奇书又得到我的真传，武功盖世加上自己的打狗棒法，帮帮打狗，招招毙命。妖魔鬼怪都怕她不得，一路上也担起保护其夫的作用；秋香温柔贤惠，心细耐劳，所以行李包囊她一肩挑，从无怨言；杨贵妃习惯了宫廷里锦衣玉食，从来都是别人伺候她，所以养的白白胖胖，耳朵大显得很有福气，睡觉是她最拿手的，对为夫倒也忠诚耿直；­

但后来不知道为何老吴把我写成唐和尚，众老婆更惨，黄蓉写成猴子，秋香写成沙和尚，我最亲爱的贵妃娘娘也被写成最恶心的猪。最最可气的是把俺三位娇妻全部写成男性。。。。。真想给他两逼斗，回想：“哎！算了，和一个陌生人计较个啥？把他拉进黑名单，一了百了。”­

一路上是爬山涉水，翻山越岭，艰难险阻，风雨兼程，妖魔鬼怪；全凭我大老婆的打狗棒和九阴白骨爪使得妖魔无一活命，那打斗的场景个个惊心动魄，不容细说。单是讲铁扇公主是如何为俺红杏出墙，牛魔王又是如何被俺绿帽盖顶，就能讲它个三天三夜。。。。。

功夫不负有心人，这不年底我们妻妾四人已经兵临天竺国城下。远望西天，云雾弥漫，一个巨大的岛屿穿插云霄之巅。周围云山云海，奇峰乱石散布于岛屿两侧但井然有序。突然那岛屿复活了，曰：­

“殿下可是自东土大唐而来，向我西天求取经文的亮仔？”­

我先是一愣再一惊，慢慢地，慢慢地云雾开始散开，我渐渐的看清楚了，那哪里是什么岛屿怪石，原来是活生生的巨人，而两侧畸形怪胎也均是人也；突然我注意到在巨人左侧上方有一位年轻身着白衣绸缎的女子。她浮云半空，长发飘飘，俊俏的脸庞带有几丝微笑，这下我完全看清楚了，这不是让我朝思暮想，夜不能寐的观音姐姐吗？哦！！是了，指定是她，我不禁喊出：­

“姐姐我在这里。”全然忘记了场合，更于如来于不顾。“我终于见到你了，我想死你了。”我大声高呼着，右臂来回摆动，且一直跳上跳下可也难以完全涌出我激动的心。­

其实并非我自作多情，观音姐姐对我也情有独钟，她一个黄花大闺女潜在一群秃驴中间，假装修行是为了混口饭吃。之中的痴情就不用多说了，目的就是为了等待这一天，看到我的到来再也压抑不住多日来心中的泪水任它打在自己尚未送给我的情书的信封上，可看到我后面有三位美女相伴这才稍安勿躁稳了情绪，没有像我一样目无佛祖。­

佛祖果然慈悲为怀没有怪罪，并很快把各种经书交付于我，看的出来此庙不留人，我鼓起勇气向佛祖要人，他却道：“此乃佛门重地，容不得你撒野，你前来是求取经书还是求取观音？观音乃我座下弟子岂能你说要就要？？说毕，右手指来回捏动着，做点钞状。­

呵呵！！俺明白她的意思，他是想要点好处。谁说和尚不贪财？？我看他穿的破旧便道：­

“我大唐以盛产丝绸和珠宝而闻名世界，以丝织成的袈裟晶莹剔透，如果加以用宝石镶嵌的话定能吸收天之灵气，地之精华，更是佛光四射。。。。。。”­

佛祖腰杆子一下子那叫一个挺拔。­

“难得你对本座一篇孝心，，，也能看的出来你为取得真经不远万里迢迢。历经九九八十一难，得不到真经誓不罢休的坚强意志，让本座和各位弟子为之感动。观音你带走吧！！其实她在挺碍事的，这里全是潜心修炼的男人，面前有这么一个前挺后翘的美丽女孩难免走神，，，难以修成正果。弟子们有时想换件衣服都难。”­

我听了连声道：­

“佛祖说得极是”­

观音的身世其实极为可怜！！！！！

我悄悄的给大家透露一下，大家要帮俺保守这个秘密哦 ！！呵呵

观音其实是如来和王母偷情所生的私生女。王母深知“要想人不知，除非己莫为”“纸里包不住火”的道理，怕她长大以后真相大白于天下，危及到自己的后位，所以不肯将其抚养，狠心抛弃，没有尽道母仪天下的责任；如来无奈才慈悲为怀，向世人慌称观音乃人间一遗孤婴，难以保命，便收为座下弟子，伴之左右。­

那时候如来弟子也不过区区数十人，但随着观音渐渐长大成人，美貌显露已经无法形容，倾倒三界众生芸芸，身体也日渐丰盈。这也不是什么秘密，世人皆知。从此佛门弟子迅速攀升，从西天到大唐摇摇万里。数国百姓抛妻弃子陆续剃度为僧，诵经念佛。­

“盼早日，修正果。”­

“赴西天，看观音。”­

整天，色即是空。­

其实，空即是色。­

无一不在打着自己心里的算盘。烧香念佛是假，盼见观音为真。可怜观音唯爱独我。­

事情是这样的，看着观音一天天长大，转眼就到谈婚论嫁的年龄，如来便派亲信到世间寻找姻缘，结果选中了我。与如来掐指神算居然不谋而合。又以去东土大唐寻找取经人为由任命于观音，有意安排我们相识，毕竟现在提倡自由婚姻。­

唉！！可怜天下爸爸心。­

观音在东土大唐天安门广场上空浮云漫漫，恰巧看到一个美男子正在晨练—无形太极。那个人就是我，后来得知我能文能武，满腹经纶，便心存爱恋，暗自发誓这辈子非我不嫁。便去向唐国皇帝言意。却碰和珅参我，她暗中点化，这才帮我解了围。太宗知道观音看上俺了才于我结拜，是想将来沾俺的佛光。­

从那以后，观音到处送礼，施舍，济贫，救民于水生火热之中。为我打通道路，树立形象。排忧解难。横扫障碍，盼与我早日重逢；同时取经还是佛祖对未来女婿的一个实地考验，后来觉得我是可靠的男人，才把爱女交给我管理。。。。。不好意思！！呵呵 。。用词不当。­

诗曰：“寒冬时节雪纷纷，路上行人全断魂。”­

班师回朝恰巧寒冬腊月，天寒地冻，雪花如同与时间赛跑。­

小时候！家境贫！遇风寒！险丧命！年累月！固成疾！寒冬至！病复发！­

众妻妾看我这般难忍，，，，，，，，，，­

不知道用什么词语形容俺当时的那个难受的滋味，，读者可完形填空，可畅所欲言。­

反正众妻妾为我操碎了心。围成一团，把我裹在其中，那真叫一个严实，用自己的体温为其夫续命，齐高呼： “只有老婆在，在寒也能睡”­

不知道什么时候开始我心里某个角落的雪静悄悄的融化成了暖暖的水暖了心。­

就这样妻子门白天如衣，夜晚为被。从此严寒再也没有来袭。安然的回到了京城。­
