想念妈妈

发表于 2018-11-02 | 分类于 Essay | | 浏览 次 | 字数： 1.5k
阅读时长 ≈ 5

这是一个山雪欲来风满楼的季节，
寒冬腊月，数九寒天的夜晚。忙活了半天的我，终于喝上了亲手熬的菜饭，是多么的欣喜。菜饭是我家乡特有的一种小米稀饭的叫法，和外面的小米粥完全不一样，味道也大不相同。我是喝菜饭长大的，妈妈说我好养活，不爱面条偏偏对菜饭情有独钟，对乡下人来说，也许不能天天吃面条，但天天喝菜饭那就相对容易多了。这一喝就整整16年。
常年在外漂泊，再也没有喝过妈妈亲手做的菜饭，一直是我的遗憾。厌烦了外面的小米粥难吃至极，同样是小米加开水，为什么差别如此之大呢？我不明白，偶尔有机会也自己熬菜饭可是总是没有妈妈熬的好喝，难道是炭火和电磁炉的火候不同造就了不同的结果？不管什么原因，我是多么希望能再多喝几年妈妈熬的稀饭，是那么的好喝，香纯可口，舒服愉悦 ， 如饮甘露，六神送爽。
想喝家乡的菜饭不禁想起来妈妈。一个善良慈祥，和蔼可亲，含辛茹苦，度十年如一日，勤俭持家的伟大的母亲。甘愿把自己一生的幸福都花在了无私的养育儿女的身上。耗尽了自己一生的心血全为了这个家，为了我和姐姐。妈妈虽然只是一个农村的妇女，一个瘦小的女人，但她默默的包揽了家里家外，所有的家务活和体力活和农活。一天到晚从不闲着，眼里总有干不完的活。我初中三年姐姐初中三年每天早晨4点多起床给我和姐姐热饭，曾多少时在伸手不见五指寒冬的清晨，顶着寒冷刺骨的无情的冷风。担心我怕黑步行5里的道路送我到邻村的中学，而后一个人抹黑回家，妈妈你那时怕黑吗？我想你一定和我一样害怕可是你没有一次不送，没有说过怕黑，怕冷。回家后还要扫雪，做早饭，刷锅洗碗，缝缝补补。，做午饭，刷锅洗碗，洗洗簌簌，做晚饭等我们回家吃，在刷锅洗碗，这一天就算您熬下来了。做饭洗碗这本不是多么幸苦的事，但是家里的条件，寒冬的季节，转出被窝就算是出了大门站到了街上，基本没有什么区别，您一天到晚做着这些反反复复事是多么的幸苦。你不烦躁吗？没有想过逃避吗？如此的坚持不懈，这些活本身就没有多大乐趣，您是在熬，这一熬就熬了一辈子，耗尽了您一生。这是多么痛苦，需要有多大的毅力？。而且天天如此？正因为如此妈妈的十根指头和手背上，密密麻麻的都是干活磨出的老茧和冻裂的缝，裂缝深的指肉都暴露了出来，几乎看见了骨头。我看见都觉得很疼，您也疼的厉害，有时疼的都掉下了泪水。您只是用白色的胶布缠着，缠满了10根指头。而这些活您还是接着做。天天做。妈妈的日子过的苦，不像别的女人那样一天到晚麻将桌上，嘻嘻哈哈过！！然而对您来说，冬天是最清闲的季节。。春天里的菜地里，田地里辛勤的播种，夏天顶着炎炎烈日锄草，秋天里您又忙着和爸爸推着平车去田里收成。哪里有活您就在哪里出现。您的勤苦，一心为家，无怨无悔的生活作风无人能及，现代女性更不能相比，所谓的劳动楷模全是扯淡。
此时我病了，眼睛生疼，虚汗和水一样往外流。我静静的躺着，闭上眼睛忍着，裹紧了被子，任凭汗水浸透了被子。我知道只要好好的出一场大汉，一觉醒来就好了，我现在要做的就是强迫自己入眠。这是妈妈告诉我的，从小就多病的我，每次病了就大汗淋漓，妈妈总是给我盖了被子，也会出一身的汗，太热了 。而我觉得掀了被子舒服点，妈妈不让，在我耳边唠唠叨叨告诉我，出一身汗，睡一觉就好了，并不时的给我盖严实生怕我中风。而现在，妈妈不在我身边，我遵循妈妈长久以来一直对我嘱咐最多的话：“把自己裹起来”任汗水往外流，泪水更是不争气的比汗水更淋漓。我想家了，想家里此时正在干活的妈妈 。想起当年为了蹒跚而行，依然离开家时，妈妈的眼湿湿的。可自己以为终于可以天高任鸟飞，海阔凭鱼跃。还是高高兴兴的拿上简单的行李启程北上，来到这个陌生的多年都无法熟悉的城市，像机器一般的为别人服务。想到家的温暖，想起有妈妈照顾的幸福感。原来我和妈妈之间有一根无形的线接着彼此，我也不曾逃离妈妈的掌心，我天南海北，妈妈的心也一直随我海角天涯。
妈妈这个词是多么的亲切，在妈妈的眼里我永远是个孩子，我喜欢这个词。不想长大，不想让妈妈老去，更没有妈妈那样强大的心灵。没有妈妈的照顾，没有妈妈的支撑我的心早已奔溃于无情的现实。妈妈你还好吗？
